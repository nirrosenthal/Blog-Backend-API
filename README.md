## Running:
1. Go to root diectory of the project

2. Build MongoDB and Flask Images:
```bash
docker-compose build
```
3. Run Docker Compose container:
```bash
docker-compose up -d
```
4. Stop Container Run:
```bash
docker-compose down
```
